# 说明

本程序主要是便于将自己的文件上传到FTP服务器，运行setup.bat就能安装，然后运行bin目录下的start.bat就能运行程序，之后只需要将要上传的文件放到upload文件夹中，文件就会自动上传，上传的文件会重命名，通过`https://aliyun.hellozjf.com:7004/uploads/重命名过的文件名`进行访问
